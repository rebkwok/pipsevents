Pop-up notification

This notifcation appears as a popup for every new session. On dismissal, it stores a cookie in the user's browser and will not be shown again unless cookies are cleared. You can specify a timeout (in seconds) if you want the popup to be shown again after a delay.

Example: You want to tell users about a new feature on the site (e.g. they can now add pronouns to their profile). Set the notice expiry to one month form now, so it should be seen by all regular users. Leave the timeout blank so users don't see the popup after the first time.

Example: You have a sale on and you want to show a daily popup reminder about a sale discount code. Set the expiry to the end date/time of the sale, and set the timeout to 86400. Users will see the popup, dismiss it, and it won't reappear until the next day.

Note: changing the title, content or timeout in this form in any way will cause the notice to reset and it will be displayed to all users again. (It will be displayed to you when you save.)
